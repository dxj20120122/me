# 个人网站项目 - 科技感设计

这是一个基于现代Web技术栈开发的个人网站项目，采用科技感设计风格，具有流畅的动画效果和响应式布局。

## 项目特点

- 🎨 现代科技感UI设计
- ✨ 流畅的动画效果
- 📱 完全响应式布局
- 🚀 优秀的性能表现
- 🎯 清晰的代码结构

## 技术栈

- HTML5
- CSS3
- JavaScript
- Particles.js

## 功能模块

- 自定义光标动画
- 粒子背景效果
- 故障艺术文字效果
- 打字机文字效果
- 3D翻转卡片
- 玻璃态UI设计

## 使用说明

1. 克隆项目到本地
```bash
gh repo clone dxj20120122/me
```

2. 打开项目
```bash
cd me
```

3. 使用本地服务器运行项目（例如使用VS Code的Live Server插件 也可以直接点击index.html文件）

## 项目结构

```
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 主要脚本文件
├── particles.min.js    # 粒子效果库
└── README.md          # 项目说明文档
```
